# SUB-verse Chronicles - Game Design Document

## Concept

### Genre
Isekai Visual Novel with RPG elements

### Premise
The protagonist is transported to a fantasy world where they must navigate political intrigue, forge alliances, and uncover the mysteries of their summoning while developing unique abilities granted by the isekai transition.

## Core Gameplay Loop

1. **Narrative Progression**: Read story dialogue and make choices
2. **Character Development**: Level up stats and unlock skills
3. **Exploration**: Discover new locations and characters
4. **Decision Making**: Shape the story through meaningful choices

## Isekai Mechanics

### Stats System
- **HP (Health Points)**: Represents physical wellbeing
- **MP (Mana Points)**: Used for special abilities
- **STR (Strength)**: Physical power
- **INT (Intelligence)**: Magical aptitude
- **AGI (Agility)**: Speed and reaction
- **CHA (Charisma)**: Social influence
- **LUK (Luck)**: Affects random events

### Skill Tree
Players unlock skills through story progression and stat investment:
- **Combat Skills**: Enhance physical confrontations
- **Magic Abilities**: Unique to the isekai protagonist
- **Social Skills**: Improve dialogue options and relationships
- **Crafting**: Create items and equipment

### Inventory System
- Equipment slots (Weapon, Armor, Accessory)
- Consumable items (Potions, Scrolls)
- Quest items (Story progression)
- Materials (Crafting components)

## Narrative Structure

### Chapter System
Story divided into chapters, each with:
- Main storyline quests
- Side character routes
- Optional exploration content
- Multiple endings based on choices

### Decision Impact
**Immediate Consequences**: Some choices have instant effects
**Long-term Effects**: Decisions influence later story branches
**Relationship Tracking**: Character affection/reputation meters

### Branching Paths
- Minimum 3 major story routes
- 5+ possible endings
- Hidden routes unlocked by specific choices

## Character System

### Protagonist
- Customizable name
- Gender selection affects some dialogue
- Personality shaped by player choices
- Unique isekai ability (revealed through story)

### Companion Characters
- 6+ main characters with full routes
- Each has unique abilities and story arcs
- Relationship levels affect gameplay
- Side characters provide context and quests

### Character Sprites
- Multiple expressions per character (neutral, happy, sad, angry, surprised)
- Costume variations for story events
- Special CG images for key scenes

## World Building

### Setting: The Realm of Aethermoor
- **Capital City**: Political center, starting location
- **Forest of Whispers**: Mysterious woodland with ancient secrets
- **Mountain Citadel**: Home to warrior clans
- **Coastal Haven**: Trading port with diverse cultures
- **Ruined Temple**: Dungeon-like exploration area

### Isekai Elements
- Protagonist has knowledge from modern world
- Status screens visible to protagonist only
- Leveling system operates like an RPG
- Magic system follows game-like rules

## UI/UX Design

### Main Menu
- New Game
- Continue
- Load Game
- Gallery (unlocked CGs)
- Settings

### In-Game HUD
- Dialogue box with character name
- Text with typing animation
- Choice buttons (when applicable)
- Quick menu (Save, Load, Log, Auto, Skip)
- Stats display (toggleable)

### Touch Controls
- Tap: Advance dialogue
- Swipe up: Open quick menu
- Swipe down: Hide UI
- Long press: Open backlog
- Pinch: Zoom CG images

## Save System

### Features
- Auto-save at decision points
- Manual save slots (20+)
- Quick save/load
- Save includes: story position, stats, inventory, flags

## Audio Design

### Music
- Main theme
- Location-specific BGM (5+ tracks)
- Character themes
- Battle music
- Emotional scene music

### Sound Effects
- UI interactions (clicks, hovers)
- Dialogue blips (character voices)
- Action sounds (magic, combat)
- Ambient environmental sounds

## Progression Systems

### Experience & Leveling
- Gain XP through story progression
- Level up grants stat points to distribute
- Skills unlock at specific levels or through quests

### Achievement System
- Story milestones
- Relationship achievements
- Hidden discoveries
- Multiple playthrough rewards

## Monetization (Future)

### Free to Play Model
- Base story free with ads
- Premium unlock removes ads
- Optional DLC chapters
- Cosmetic character sprites

### Premium Model (Alternative)
- One-time purchase
- All content included
- Free updates

## Accessibility Features

- Text size adjustment
- Text speed control
- Auto-advance mode
- Skip read dialogue option
- Color blind friendly UI
- Screen reader support (future)

## Development Priorities

### MVP (Minimum Viable Product)
- Chapter 1 complete (30-60 minutes gameplay)
- 3 main characters introduced
- Basic stats and inventory system
- Core visual novel functionality
- 1 branching path with 2 endings

### Full Release
- Complete story (8-12 hours)
- All 6+ character routes
- Full skill tree and progression
- Gallery and achievement system
- Polished UI and animations
